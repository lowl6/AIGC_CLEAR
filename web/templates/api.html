TruthLens API 文档

通过 RESTful API 将 AI 内容检测与传播风险评估能力集成到您的系统中。支持文本、图片、链接和批量检测。

简单集成
RESTful 接口，JSON 格式，5分钟完成对接
毫秒级响应
平均响应时间 < 2s，99.9% 可用性 SLA
安全可靠
HTTPS 加密传输，数据不留存
Base URL https://api.truthlens.cn/v2

认证方式

所有 API 请求需要在 Header 中携带 API Key 进行认证：

HTTP Header
Authorization: Bearer YOUR_API_KEY
安全提示：请勿在前端代码中暴露 API Key。建议通过后端服务转发 API 请求。

快速上手

3 步完成你的第一次 API 调用：

1
获取 API Key

在控制台创建应用并获取 API Key

2
发送检测请求

调用对应端点提交需要检测的内容

3
获取检测结果

实时返回 AI 生成概率和传播风险等级

Python 示例 
import requests

# 文本检测 API 调用
response = requests.post(
    "https://api.truthlens.cn/v2/detect/text",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "text": "这款面霜真的太好用了！作为一个干皮星人...",
        "platform": "xiaohongshu",
        "options": {
            "deep_analysis": True,
            "risk_assessment": True
        }
    }
)

result = response.json()
print(f"AI 生成: {result['ai_generated']}")
print(f"概率: {result['ai_probability']}")
print(f"风险等级: {result['risk_level']}")
POST /detect/text

文本检测

检测输入文本是否由 AI 生成，并评估传播风险等级。

请求参数

参数 类型 必填 说明
text string 是 待检测的文本内容，1~10000 字符
platform string 否 来源平台：xiaohongshu / douyin / taobao / weixin / other
options.deep_analysis boolean 否 启用段落级深度分析，默认 false
options.risk_assessment boolean 否 启用传播风险评估，默认 true
options.language string 否 语言：auto / zh / en，默认 auto

POST /detect/image

图片检测

检测图片是否由 AI 生成（Midjourney, DALL·E, Stable Diffusion等），同时可 OCR 提取图上文字进行分析。

参数 类型 必填 说明
image file / base64 是 图片文件或 Base64 编码，最大 20MB
ocr boolean 否 是否 OCR 提取图片文字，默认 true
platform string 否 来源平台标识
POST /detect/url

链接检测

提交平台内容链接，系统自动抓取并分析内容。支持小红书、抖音、淘宝、微信公众号等主流平台。

参数 类型 必填 说明
url string 是 内容页面 URL
include_comments boolean 否 是否同时抓取评论区内容，默认 false
POST /detect/batch

批量检测

提交多条内容进行批量检测，返回 batch_id 用于查询结果。最多支持 500 条/次。

参数 类型 必填 说明
items array 是 检测项数组，每项含 type(text/url/image) 和 content
callback_url string 否 检测完成后回调通知地址
GET /detect/result/{request_id}

查询检测结果

根据 request_id 或 batch_id 查询检测结果。对于批量检测和异步任务，需要通过此接口轮询获取结果。

风险等级说明

等级 标识 分数范围 说明
L1 低风险 0 - 25 AI辅助创作，内容合规，传播影响可控
L2 中风险 26 - 50 AI生成痕迹明显，需标注，建议人工复核
L3 高风险 51 - 75 高度疑似AI伪造，涉及虚假种草/评测
L4 极高风险 76 - 100 确认AI伪造，涉及深度伪造/诈骗/虚假新闻
