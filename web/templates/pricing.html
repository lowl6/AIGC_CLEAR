常见问题

免费版有什么限制？

免费版每日可进行 100 次文本检测和 20 次图片/链接检测，使用基础检测模型，不支持段落级深度分析和 API 接入。适合个人体验和轻度使用。

API 版和专业版有什么区别？

专业版适合通过网页界面进行检测的个人用户和自媒体从业者。API 版面向需要系统集成的开发者和企业，提供 RESTful API 接口、多语言 SDK 和 Webhook 回调。

检测数据是否会被保留？

免费版和专业版的检测数据保留 30 天。API 版不保留请求数据，仅保留调用日志。企业版可按需配置数据保留策略。

支持哪些平台的内容检测？

目前支持小红书、抖音、快手、淘宝、京东、拼多多、微信公众号、微博等主流平台的内容检测。链接检测会自动识别平台类型并优化检测策略。
